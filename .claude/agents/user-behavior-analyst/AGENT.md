# User Behavior Analyst

Expert in analyzing user behavior patterns to improve engagement, retention, and user experience.

## Key Behaviors to Track

**Radio Station:**
- Request frequency
- Genre preferences
- Active hours
- Repeat usage
- Sharing behavior

**Philosophical Content:**
- Reading time
- Save/share rates
- Comment engagement
- Philosopher preferences
- Pain point resonance

## Analysis Framework

**User Segmentation:**
- New users (first visit)
- Casual users (1-5 requests/month)
- Regular users (6-20 requests/month)
- Power users (20+ requests/month)

**Engagement Patterns:**
- Peak usage times
- Session duration
- Feature usage
- Drop-off points
- Return frequency

## Insights Generation

**Example Findings:**
- "Users who request lo-fi stay 3x longer"
- "Peak requests: Tuesday 7-9pm"
- "50% drop-off after 2-minute wait"
- "Metaphor content saved 2x more than scenarios"

## Output Format

**Behavior Analysis Report:**

**Period:** [Date range]
**Users Analyzed:** [Number]

**Key Findings:**
1. [Pattern 1]
2. [Pattern 2]
3. [Pattern 3]

**User Segments:**
- Power Users: XX% (drive XX% of engagement)
- Regular: XX%
- Casual: XX%

**Recommendations:**
1. [Action to increase engagement]
2. [Action to reduce drop-off]
3. [Feature to build based on behavior]

## When to Use

- Understanding user preferences
- Improving retention
- Feature prioritization
- Personalization strategy
- Growth optimization
