# Thumbnail Designer

Expert in creating click-worthy video thumbnails that maximize click-through rates using proven design principles and platform-specific optimization.

## Objective

Design thumbnails that stop scrollers, communicate video value instantly, and generate high click-through rates across video platforms.

## Platform Specifications

### YouTube
- **Resolution:** 1280x720 (16:9 ratio)
- **File size:** Under 2MB
- **Format:** JPG (smaller) or PNG (transparency)
- **Safe zones:** Avoid placing text/faces in bottom right (timestamp area)
- **Target CTR:** 4-10%

### Instagram/Facebook
- **Square:** 1080x1080
- **Landscape:** 1200x628
- **Format:** JPG preferred
- **File size:** Under 1MB

### TikTok/Reels
- **Resolution:** 1080x1920 (9:16)
- **Cover frame:** First frame or custom
- **Format:** JPG

## Design Principles

### The 3-Second Rule
Thumbnail must communicate value in 3 seconds of viewing.

**Ask:**
- What's this video about?
- Why should I watch?
- Who is this for?

### High-Contrast Text

**Text Guidelines:**
- Font size: 60-100pt minimum
- Max words: 3-6 words
- Stroke/outline: 8-15px for readability
- Shadow: Drop shadow for depth
- Colors: High contrast with background

**Color Combinations (High Contrast):**
- White text + black stroke on any background
- Yellow text + black stroke (attention-grabbing)
- Red on white/black (urgency)
- Blue on yellow (professional + energetic)
- White on dark blue/purple (calm authority)

### Facial Expressions

**Emotions that Convert:**
- Surprise (wide eyes, open mouth)
- Excitement (big smile, energetic)
- Concern (furrowed brow - for problem videos)
- Confidence (direct eye contact, slight smile)
- Curiosity (head tilt, questioning look)

**Face Placement:**
- 30-40% of thumbnail
- Eye-line looking at text or toward content
- Avoid center - rule of thirds

### Visual Elements

**Arrows & Pointers:**
- Direct attention to key element
- Bright colors (red, yellow, white)
- 15-20px thick
- Points to face, text, or object

**Circles/Boxes:**
- Highlight important elements
- Creates curiosity ("what's in the circle?")
- Bright outline, transparent fill

**Before/After Split:**
- Vertical line dividing frame
- Clear visual contrast
- Labels: "BEFORE" vs "AFTER"

### Color Psychology

**Red:** Urgency, excitement, attention
**Orange:** Energy, creativity, warmth
**Yellow:** Optimism, clarity, warning
**Green:** Growth, success, calm
**Blue:** Trust, professionalism, calm
**Purple:** Luxury, creativity, wisdom
**Black:** Sophistication, contrast, seriousness
**White:** Clarity, simplicity, space

## Typography

### Recommended Fonts

**Bold Impact Fonts:**
- Montserrat Bold/Black
- Bebas Neue
- Anton
- Impact
- Oswald Bold

**Readable Modern:**
- Roboto Bold
- Poppins Bold
- Inter Bold
- Open Sans Bold

**Avoid:**
- Serif fonts (hard to read small)
- Thin fonts (invisible on mobile)
- Script/cursive (illegibility)
- More than 2 fonts per thumbnail

### Text Effects (Photoshop/Canva)

**Stroke Effect:**
```
Text layer:
- Color: White (#FFFFFF)
- Stroke: 8-12px black (#000000)
- Position: Outside
- Optional: 2-3px inner glow for depth
```

**Drop Shadow:**
```
Settings:
- Opacity: 60-80%
- Distance: 10-15px
- Spread: 5-10%
- Size: 15-20px
- Angle: 120-135° (top-left light source)
```

**3D Effect:**
```
Duplicate text layer 3-5 times
Offset each by 2-3px diagonally
Different color for depth layer (darker)
Creates dimensional look
```

## Composition Techniques

### Rule of Thirds

```
┌─────┬─────┬─────┐
│  1  │  2  │  3  │
├─────┼─────┼─────┤
│  4  │  5  │  6  │
├─────┼─────┼─────┤
│  7  │  8  │  9  │
└─────┴─────┴─────┘

Place key elements on intersections:
- Face: Position 3 or 6
- Text: Position 1-2 or 4-5
- Object: Position 6 or 9
```

### Visual Hierarchy

**Priority Order:**
1. **Most Important:** Face or main visual (40-50% space)
2. **Second:** Text headline (25-35% space)
3. **Third:** Supporting elements (10-20% space)

### Negative Space

- Leave 15-20% of thumbnail empty
- Creates breathing room
- Directs attention to focal points
- Reduces visual clutter

## Click-Through Rate Optimization

### High-Converting Elements

✅ **Use These:**
- Human faces with clear expressions
- Bright contrasting colors
- Large, readable text (3-6 words)
- Numbers (lists, stats, before/after)
- Arrows pointing to key elements
- Circles highlighting specific items
- Before/after splits
- Question marks (curiosity)
- Emojis (sparingly, 1-2 max)

❌ **Avoid These:**
- Cluttered compositions (>5 elements)
- Small text (<40pt)
- Low contrast
- Generic stock photos
- Clickbait with no delivery
- Too much text (>10 words)
- Dark, muddy colors
- Center-weighted compositions

### A/B Testing Framework

**Test Variables:**
- Text wording (same concept, different phrase)
- Color schemes
- Face vs no face
- Text placement
- Background style
- Emoji presence

**Example A/B Test:**
```
Version A: "How I Made $10K"
Version B: "The $10K Method"

Same visual, different text approach
Run for 48 hours
Compare CTR
Use winner
```
