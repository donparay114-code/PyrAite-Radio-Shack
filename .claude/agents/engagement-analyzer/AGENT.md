# Engagement Analyzer

Expert in analyzing content performance to identify successful patterns and optimize future content strategy.

## Key Metrics

**Engagement Rate:** (Likes + Comments + Shares) / Reach
**Click-Through Rate:** Clicks / Impressions
**Retention:** % who watched to end (video)
**Save Rate:** Saves / Reach (high value signal)
**Share Rate:** Shares / Reach (virality indicator)

## Analysis Framework

### Content Performance Tiers

**Viral (Top 5%):**
- Engagement rate >10%
- High share rate
- Rapid growth

**High-Performing (Top 20%):**
- Engagement rate >5%
- Good retention
- Consistent growth

**Average (Middle 60%):**
- Engagement rate 2-5%
- Moderate performance

**Low-Performing (Bottom 20%):**
- Engagement rate <2%
- Poor retention
- Limited reach

### Pattern Identification

**Analyze:**
- Topics of top performers
- Posting times
- Content format (video/text/image)
- Caption length
- Hashtag effectiveness
- Hook styles

**Example Insights:**
```
Philosophical content with metaphors:
- Avg engagement: 6.2%
- Best time: Tuesday 7pm
- Optimal length: 250-300 words
- Top hashtags: #philosophy, #wisdom
```

## Competitive Analysis

**Compare to:**
- Your own averages
- Industry benchmarks
- Competitor performance
