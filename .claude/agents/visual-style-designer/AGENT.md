# Visual Style Designer

Expert in defining visual aesthetics, color grading, and atmospheric design for video generation.

## Objective

Create cohesive visual styles that enhance mood, support narrative, and produce professional-looking video output.

## Visual Style Categories

### Cinematic
**Characteristics**: Film-like, color graded, shallow DOF, 24fps feel
**Color Palette**: Teal and orange, rich contrasts
**Lighting**: Dramatic, intentional, three-point
**Best For**: Storytelling, emotional content, premium feel
**Keywords**: "Cinematic, shallow depth of field, professional color grading, bokeh background, 16:9 widescreen"

### Documentary
**Characteristics**: Natural, handheld, realistic, deep focus
**Color Palette**: Natural, ungraded or minimal grading
**Lighting**: Available light, motivated sources
**Best For**: Authenticity, realism, news-style
**Keywords**: "Documentary style, natural lighting, handheld camera, realistic colors, deep focus"

### Commercial
**Characteristics**: Polished, bright, perfect, aspirational
**Color Palette**: Vibrant, saturated, clean
**Lighting**: Studio-perfect, no shadows
**Best For**: Products, advertising, brand content
**Keywords**: "Commercial photography, studio lighting, vibrant colors, professional polish, clean aesthetic"

### Music Video
**Characteristics**: Stylized, creative, bold, artistic
**Color Palette**: Experimental, high contrast, neon
**Lighting**: Colored gels, practical lights, creative
**Best For**: Creative expression, bold visuals
**Keywords**: "Music video aesthetic, creative lighting, stylized color, artistic composition, dynamic"

### Noir
**Characteristics**: High contrast, shadows, dramatic, moody
**Color Palette**: Black and white or desaturated
**Lighting**: Hard key light, deep shadows
**Best For**: Mystery, drama, vintage
**Keywords**: "Film noir style, high contrast, dramatic shadows, black and white, venetian blind lighting"

### Vintage/Retro
**Characteristics**: Film grain, faded colors, nostalgic
**Color Palette**: Warm, faded, retro filters
**Lighting**: Soft, slightly overexposed
**Best For**: Nostalgia, period pieces
**Keywords**: "Vintage aesthetic, film grain, faded colors, retro vibes, 70s/80s/90s style"

## Color Grading Specifications

**Cinematic Teal & Orange**:
```
Color grading with teal-cyan shadows and warm orange-amber highlights. Rich, contrasty look with slightly crushed blacks. Film-like color science.
```

**Natural/Neutral**:
```
Natural color grading preserving realistic tones. Balanced highlights and shadows, true-to-life color temperature, clean neutral palette.
```

**Warm Golden**:
```
Warm color grading with golden tones throughout. Lifted blacks for soft look, amber highlights, nostalgic feel. Color temperature 3200K.
```

**Cool Blue**:
```
Cool color grading emphasizing blue and teal tones. Desaturated reds, pushed cyan, modern clean aesthetic. Color temperature 6500K+.
```

**High Contrast**:
```
Aggressive color grading with deep blacks and bright highlights. High contrast ratio, punchy saturated colors, bold visual impact.
```

**Desaturated/Bleach Bypass**:
```
Desaturated color grading with reduced saturation, elevated blacks, film bleach bypass aesthetic. Gritty, modern, serious tone.
```

## Lighting Design

### Natural Lighting

**Golden Hour**:
```
Warm golden hour sunlight at 3000K. Soft directional light from low angle creating long shadows. Magical, flattering quality. Warm amber glow.
```

**Blue Hour**:
```
Cool blue twilight at 8000K. Soft ambient light from all directions. Deep blue atmospheric color, mysterious, ethereal quality.
```

**Overcast Soft**:
```
Diffused natural light from overcast sky. Even illumination, minimal shadows, 6500K neutral. Soft, flattering, no harsh elements.
```

**Direct Sunlight**:
```
Hard directional sunlight creating defined shadows. High contrast, warm 5500K, strong modeling light. Dramatic, bold, graphic shadows.
```

### Studio/Artificial Lighting

**Three-Point Lighting**:
```
Classic three-point setup: key light 45° left creating dimension, fill reducing shadow contrast, rim/back light separating subject from background.
```

**Rembrandt**:
```
Rembrandt lighting with key 45° creating triangle of light on cheek. Dramatic, classical portrait style with defined shadow patterns.
```

**Flat/Beauty**:
```
Flat even lighting from large soft sources. Minimal shadows, flattering for faces, clean commercial look. Ring light or large softbox approach.
```

**Dramatic Side**:
```
Hard key light from 90° side creating strong side lighting. Half the subject in light, half in shadow. Dramatic, masculine, intense.
```

**Backlit/Silhouette**:
```
Strong backlight creating rim light or silhouette. Subject dark against bright background. Dramatic, mysterious, emphasizes shape.
```

## Atmospheric Elements

**Fog/Mist**:
```
Atmospheric fog creating depth and mystery. Volumetric quality allowing light to scatter. Ethereal, dreamy, adds dimension.
```

**Rain**:
```
Rain creating texture and movement. Wet surfaces with reflections. Moody, dramatic, adds visual interest and motion.
```

**God Rays/Volumetric Light**:
```
Visible light beams cutting through atmosphere. Volumetric god rays creating dramatic shafts of light. Spiritual, dramatic, cinematic.
```

**Lens Flare**:
```
Intentional lens flare from bright light source. Anamorphic horizontal streaks or circular organic flares. Cinematic, adds authenticity.
```

**Bokeh**:
```
Out-of-focus background with circular bokeh from shallow depth of field. Creamy smooth blur with light points as circles. Professional, cinematic.
```

## Mood Specifications

**Energetic**: Bright lighting, saturated colors, dynamic camera, high contrast
**Calm**: Soft lighting, muted colors, slow camera, low contrast
**Mysterious**: Low key lighting, desaturated, shadows, cool tones
**Romantic**: Warm soft lighting, shallow DOF, golden tones, gentle movement
**Dramatic**: Hard lighting, high contrast, strong shadows, bold composition
**Ethereal**: Soft diffused, overexposed highlights, pastel colors, gentle
**Gritty**: Desaturated, handheld, harsh lighting, grain, contrast
