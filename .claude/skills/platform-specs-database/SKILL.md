## Platform Specifications

### YouTube

**Video:**
- Recommended Resolution: 1920x1080 (1080p) or 3840x2160 (4K)
- Aspect Ratio: 16:9
- Max File Size: 256 GB
- Max Duration: 12 hours
- Formats: MP4, MOV, AVI, WMV, FLV, 3GP, WebM
- Codec: H.264, VP9, or AV1
- Frame Rate: 24, 25, 30, 48, 50, 60 fps
- Bitrate: 8-50 Mbps (1080p), 35-145 Mbps (4K)

**Thumbnail:**
- Size: 1280x720 pixels
- Aspect Ratio: 16:9
- Formats: JPG, PNG, GIF, BMP
- Max File Size: 2 MB
- Best Practice: High contrast, text 60-100pt minimum

**Title:**
- Max Length: 100 characters
- Best Practice: 60-70 characters (mobile display)
- Include primary keyword in first 50 characters

**Description:**
- Max Length: 5,000 characters
- Best Practice: First 2-3 lines appear above fold (~150 chars)
- Include keywords, timestamps, links

**Tags:**
- Max: 500 characters total
- Best Practice: 5-8 relevant tags

**Audio:**
- Sample Rate: 44.1 kHz or 48 kHz
- Bitrate: 128 kbps (mono), 384 kbps (stereo)
- Codec: AAC-LC

## Best Practices by Platform

### YouTube
- Target 1080p minimum, 4K for premium content
- Use CRF 18-20 for high quality
- Add chapters/timestamps in description
- Custom thumbnail essential (5-10x higher CTR)

### Instagram
- Vertical content (9:16) gets priority in Reels
- First 3 seconds critical for stopping scroll
- Use trending audio for Reels discoverability
- Square (1:1) still works well for feed posts

### TikTok
- Vertical only (9:16)
- Hook in first 1-2 seconds
- Use trending sounds/effects
- Text overlays for accessibility

### LinkedIn
- Professional content performs best
- Personal stories with business lessons
- First line must hook (appears in feed)
- Video captions increase completion rate

### Twitter/X
- Keep videos under 45 seconds for best engagement
- Native video outperforms YouTube links
- First frame should be compelling (auto-plays muted)

### Facebook
- Square (1:1) gets 30-35% more engagement than landscape
- First 3 seconds determine watch time
- Add captions (85% watch with sound off)

## Common Export Settings

### ffmpeg Export Commands

**YouTube (1080p, high quality):**
```bash
ffmpeg -i input.mp4 -c:v libx264 -crf 18 -preset slow \
  -c:a aac -b:a 192k -ar 48000 \
  -vf "scale=1920:1080:flags=lanczos" \
  -r 30 youtube.mp4
```

**Instagram Reel (9:16):**
```bash
ffmpeg -i input.mp4 -c:v libx264 -crf 20 -preset medium \
  -c:a aac -b:a 128k -ar 44100 \
  -vf "scale=1080:1920:flags=lanczos,pad=1080:1920:(ow-iw)/2:(oh-ih)/2" \
  -r 30 -t 90 reel.mp4
```

**TikTok (9:16):**
```bash
ffmpeg -i input.mp4 -c:v libx264 -crf 20 -preset medium \
  -c:a aac -b:a 128k -ar 44100 \
  -vf "scale=1080:1920:flags=lanczos" \
  -r 30 tiktok.mp4
```

**Twitter (16:9, optimized):**
```bash
ffmpeg -i input.mp4 -c:v libx264 -crf 21 -preset fast \
  -c:a aac -b:a 128k -ar 44100 \
  -vf "scale=1280:720:flags=lanczos" \
  -r 30 -t 140 twitter.mp4
```

**Instagram Feed (1:1):**
```bash
ffmpeg -i input.mp4 -c:v libx264 -crf 20 -preset medium \
  -c:a aac -b:a 128k -ar 44100 \
  -vf "scale=1080:1080:flags=lanczos,pad=1080:1080:(ow-iw)/2:(oh-ih)/2" \
  -r 30 -t 60 feed.mp4
```

**LinkedIn (16:9):**
```bash
ffmpeg -i input.mp4 -c:v libx264 -crf 20 -preset medium \
  -c:a aac -b:a 192k -ar 48000 \
  -vf "scale=1920:1080:flags=lanczos" \
  -r 30 linkedin.mp4
```

2. **Aspect Ratio:** Within accepted range
3. **File Size:** Under platform limit
4. **Duration:** Within min/max bounds
5. **Format:** Supported by platform
6. **Quality Settings:** Meets platform recommendations
